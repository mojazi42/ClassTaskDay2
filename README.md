# Kotlin Playground

This repository contains various Kotlin examples, including:
- Kotlin data types
- Lists, Sets, Maps
- Control flow (if-else)
- Functions and Lambda expressions

## How to Run the Code
1. Clone the repository: 
https://github.com/mojazi42/ClassTaskDay2.git

2. Open in **Android Studio**.
3. Run `Main.kt`.
